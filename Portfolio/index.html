Philosophy

◆Motto

If you want to go fast, go alone. If you want to go far, go together.

私のアルバイト先は日々約15人ほどのスタッフがチームとして店舗運営しています。 そこで自らがリーダーとして店舗運営する際に、個人スキルよりもチームによる運営の方が可能性が広がる事を学びました。 目の前の小さな問題を解決する時は個人スキルの方が有利かもしれません。 しかし、より大きな目標を解決するにはチームで挑んだ方が可能性が広がることを学びました。 店舗としてより大きな目標を達成するために、多くの人を巻き込みながらスタッフを牽引してきました。

◆Motivation

自分の中のときめきやワクワクに素直に

小さい時から映画が好きで、その時からSF映画の世界のようなものを夢見ていました。 その影響か、新しい技術や最先端テクノロジーに興味があり、ニュースや実物を見る度いつもワクワクしており、心を弾ませていました。 そして現在、そのワクワクを自らの手で作り上げることができること事態にワクワクしており、このワクワクこそが私自身のモチベーションであり軸である。

◆Motivation

行動力を武器に新たな挑戦を!

私は将来、自分が思うワクワクを具現化し、最先端テクノロジーを用いて新しいライフスタイルを創造したいです。 大学の自由課題では実店舗におけるコロナ対策を図るシステム開発をし、研究では医療分野に携わる脳波信号の解析をし、学外活動としてフリーランスとしてWebデザイナーとして活動しています。 このように、分野に縛られる事なく広い視野で社会問題に向き合い、人々がワクワクし当たり前に使われるようなソリューションを提供できるようになりたいと考えております。
